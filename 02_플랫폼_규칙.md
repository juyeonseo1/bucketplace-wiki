# 플랫폼 규칙 및 Retool 버튼 안내

## Retool 버튼 안내

| 버튼 | 의미 |
|------|------|
| 🟣 **오늘의집 상품** | 오늘의집이 최저가일 때 선택 |
| 🔵 **확인 완료** | 시스템 최저가가 정확할 때 |
| 🔵 **확인 전으로 변경** | 재검수 필요할 때 |
| 🔴 **최저가 수정하기** | 시스템 수집 오류로 직접 수정해야 할 때 |

> 시스템이 수집한 타겟 상품의 플랫폼명은 수정하지 않음
