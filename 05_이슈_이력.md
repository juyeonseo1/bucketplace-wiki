### 오매칭 세부 케이스 분류 (95건 중 실제 오매칭 77건)

| 케이스 | 건수 |
|--------|------|
| 다른 제품 | 32건 |
| 타사/타브랜드 상품 | 18건 |
| 사이즈/옵션 상이 | 13건 |
| 이미지/사진 다름 (육안 확인 필요) | 7건 |
| 구성품 상이 | 5건 |
| 전시/DP 상품 | 2건 |

- **같은 제품군 내 타입 다름 (13건):** 브랜드·제품명은 같은데 세부 타입/사양이 다른 경우
  - 예: 퓨리 밥솥형 → 스몰형 매칭, 헤이미 서랍 옷장 → 슬라이드 옷장
- **동일 카테고리 다른 제품 (13건):** 카테고리는 같지만 완전히 다른 제품이 매칭된 경우
  - 예: 딱따구리 도어벨 → 다른 도어벨, 보카시 러그 → 코지 러그
- **반복 오매칭 (5건):** 동일한 제품이 계속 같은 잘못된 타몰 상품으로 매칭되는 경우
  - 모듈가구 사이드테이블 201 → 비햅핀 직사각 사이드테이블 (2회 반복)
  - 마르체 폴드 서랍장 1200 → 2단 서랍장 틈새 (3회 반복)
